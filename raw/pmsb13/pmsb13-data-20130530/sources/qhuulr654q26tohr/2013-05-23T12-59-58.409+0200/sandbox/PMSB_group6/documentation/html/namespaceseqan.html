checks whether the new record should be assigned to the cluster (represented by clusterRecord). The function checks whether the new record should be assigned to the cluster (represented by clusterRecord). If the given score is greater or equal the threshold and if the length of the new record's sequence does not exceed (<= and >=) the length of the cluster master (+- relative lengthThreshold), the function returns true.
