Template-Spezialisierung:

String<TValue, TSpec>

Spec ist nicht nur atomar

String<Dna> myGenome; // A default string of nucleotides.
String<Dna, Alloc<> > myGenome; // The same as above.

sondern kann selbst ein Template sein (TODO: bessere Beispiel finden!)

String<Dna, Array<100> > genome

Ganze Programmkonstruktute bleiben größtenteils unverändert, wenn String<Dna> nach String<AminoAcid> geändert wird.
