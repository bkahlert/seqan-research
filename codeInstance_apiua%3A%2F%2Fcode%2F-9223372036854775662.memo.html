Beispiel 1 

Alien: Problem occurred in the index tutorial  hat konkret die Arbeit mit Index , hash und getOccurrences gemeint:

  1. Problem: hash ist ein SeqAn-Implementierungsdetail, das den Anwender nichts angeht.
  • hash speichert Ergebnis in seiner Eingabe
  • getOccurrences verwendet erhält keinen expliziten hash-Parameter (“CAT”), sondern erhält dieses über ein anderes Objekt,
    in dem dieser Parameter bereits vorliegt (“indexShape(index”)
  • Lösung: getOccurrences muss hash indirekt aufrufen
    Alternativlösung: Versteckte Parameterübergabe 

Quelle:  http://seqan.readthedocs.org/en/latest/Tutorial/IndexQGram.html  (wichtig in Fett)

 

int main()
{
    typedef Index<DnaString, IndexQGram<UngappedShape<3> > > TIndex;
    TIndex index("CATGATTACATA");
    hash(indexShape(index), "CAT");
    for (unsigned i = 0; i < length(getOccurrences(index, indexShape(index))); ++i)
      std::cout << getOccurrences(index, indexShape(index))[i] << std::endl; 
      return 0;
}
