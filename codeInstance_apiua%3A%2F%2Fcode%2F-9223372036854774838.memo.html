Wie erlaube ich API-Anwendern die Modifikation von Datenstrukturen.

Bei SeqAn:

Zwei Eigenschaften:

  • Direktheit (direkt, über Umwege)
    • direkt: x = 1, fn(x, 1), x[y] = 1
    • indirekt: fn(x) = 1, fn(fm(x, 1))
  • Explizitheit (mit Operator, mit Funktionsrückgabe)
    • explizit: x = 1,  fn(x) = 1
    • implizit: fn(x, 1), fn(fm(x, 1)
  1. fn(x, 1) - ändert x
  2. fn(x) = 1 - ändert x
  3. fn(fm(x), 1) - ändert x
  4. x = 1 - ändert x

 

  • direkt, explizit: x = 1
  • direkt, implizit: fn(x, 1)
  • indirekt, explizit: fn(x) = 1
  • indirekt, implizit: fn(fm(x, 1)

Begründung von Rahn (siehe ph): Abhängigkeiten (Technisches Wegargumentieren)
