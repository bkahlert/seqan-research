Dieser Code beschreibt Code-Änderungen spielerisch-verstehender Art. Änderungen sind Folgen von Fragestellungen wie "Was passiert, wenn ich die Änderung durchführe?" oder "Tritt Effekt X ein, wenn ich Änderung Y anwende?" Eine typische Form ist das Auskommentieren. Diese Kategorie ist nicht sehr ausgegoren; alle Phänomene müssen noch einmal näher betrachtet werden.
