Inkrementierung von Rückgaben von Funktionsaufrufen

In C++ möglich, wenn Funktion eine Referenz zurückgibt.

Beispiel: 

Zeile 18, http://docs.seqan.de/seqan/dev2/files/DEMO_Iterator+_Basics.html

seqan::Iterator<seqan::String<char>, seqan::Rooted >::Type it2 = begin(str);
for (goBegin(it2); !atEnd(it2); goNext(it2)) 
{
	++value(it2);
}
